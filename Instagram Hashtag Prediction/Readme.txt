The data set was taken from Kaggle Competition hosted by Deep Learning Class Professor

Data Description 

The files are briefly described below:
Code files
There are three iPython notebooks that show you how to train a model and create a prediction output in Kaggle format. Please run the code to understand the concepts, and modify them to build your own model. For functions of each file please read the "README.txt". The "utility.py" is used the demo code.

Data files
"harrison_features.npz" - Pre-extracted features of images in harrison dataset. We extracted ImageNet and places365 CNN features.
"tag_list.txt" - The list of hashtags for all image files.

Link to Dataset:https://www.kaggle.com/c/109-1-ntut-dl-app-hw2/data
